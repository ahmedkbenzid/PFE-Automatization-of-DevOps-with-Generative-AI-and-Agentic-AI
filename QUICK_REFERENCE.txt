┌──────────────────────────────────────────────────────────────────┐
│ 1️⃣  CHECK IF __PYCACHE__ IS IN GIT                               │
└──────────────────────────────────────────────────────────────────┘
  git ls-files | grep __pycache__
  
  ➜ Returns nothing? ✅ You're good! (Skip to #3)
  ➜ Returns files?   ⚠️  Continue to #2

┌──────────────────────────────────────────────────────────────────┐
│ 2️⃣  REMOVE FROM GIT HISTORY (⚠️ Rewrites History!)               │
└──────────────────────────────────────────────────────────────────┘
  
  Windows:     cleanup_pycache.bat
  PowerShell:  .\cleanup_pycache.ps1
  Linux/Mac:   ./cleanup_pycache.sh
  
  Then force push:
  git push origin --force --all
  
  ⚠️  WARNING: Team members must re-sync!

┌──────────────────────────────────────────────────────────────────┐
│ 3️⃣  REMOVE FROM WORKING DIRECTORY (✅ Safe!)                      │
└──────────────────────────────────────────────────────────────────┘
  
  Windows:     remove_pycache_local.bat
  PowerShell:  .\remove_pycache_local.ps1
  Linux/Mac:   find . -name __pycache__ -type d -exec rm -rf {} +

┌──────────────────────────────────────────────────────────────────┐
│ 4️⃣  VERIFY .GITIGNORE                                            │
└──────────────────────────────────────────────────────────────────┘
  ✅ Already configured:
     __pycache__/
     *.pyc
     *.pyo
     *.pyd

┌──────────────────────────────────────────────────────────────────┐
│ 🆘 TEAM RE-SYNC (After Force Push)                              │
└──────────────────────────────────────────────────────────────────┘
  git fetch origin
  git reset --hard origin/main
